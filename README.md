# cursolibertaas
O Curso Libertaas é uma jornada de transformação emocional baseada na minha própria história. Em aulas curtas e envolventes, você compreenderá a origem das suas dores, fortalecerá sua identidade e romperá ciclos que limitam sua vida. Acesso por 1 ano. **Mulheres Curadas. Filhos Livres. Gerações Transformadas.**
